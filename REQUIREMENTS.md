# Requirements

## Platform
- n8n (self-hosted or n8n Cloud)

## Credentials Needed
- Telegram Bot Token (from [@BotFather](https://t.me/BotFather) on Telegram)
- Telegram Chat ID (from [@userinfobot](https://t.me/userinfobot) on Telegram)

## APIs Used
- Open-Meteo Weather API is free, no API key required
  - https://open-meteo.com

## Files in This Repo
- `weather-update-automation.json` the exported n8n workflow, ready to import
- `README.md` project overview and setup instructions
